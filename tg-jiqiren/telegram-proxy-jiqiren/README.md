## 线路失效怎么办

线路失效可能来自更新、到期、网络环境或设备兼容问题。先确认自己使用的是机器人当前提供的线路，而不是历史截图中的信息；再检查 Telegram 客户端和设备网络状态。如果机器人已经提示旧线路失效，应等待新线路，不要在公开渠道交换私人节点凭据。
