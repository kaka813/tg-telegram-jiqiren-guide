## 常见问题

### 哪个机器人适合普通用户？

如果目标是搜索 Telegram 频道、群组或相关资源，优先使用 TG云搜。转载和互推机器人主要面向频道主、管理员与运营人员。

### Telegram机器人是官方功能吗？

机器人运行在 Telegram 平台上，但具体机器人通常由第三方开发者运营。本专题与 Telegram 官方无隶属关系，也不对第三方搜索结果或频道内容作担保。

### 为什么没有收录代理、游戏或成人机器人？

本专题只聚焦搜索、转载和频道互推三类用途，保持内容主题清晰，也降低无关或高风险内容对读者判断的干扰。
