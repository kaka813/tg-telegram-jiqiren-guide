## 常见问题

### 互推是否一定能增加用户？

不能保证。效果受到受众匹配、频道内容、文案、发布时间和双方信誉影响。建议进行小规模测试，并记录结果。

### 什么是TG互推车队？

通常指多个频道按约定轮流或集中互相推广。参与前应了解规则、频率和退出方式，避免推广过密影响现有读者。

### 新频道可以马上互推吗？

可以寻找合作，但更建议先准备足够的原创内容、清晰简介和稳定更新计划，否则新用户进入后缺少继续关注的理由。
