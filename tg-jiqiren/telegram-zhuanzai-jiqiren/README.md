- 同时运营两个或多个频道的频道主。

## 如何降低自动同步风险

把“自动化”和“无人检查”区分开。重要公告、商业合作、版权敏感内容最好保留人工确认环节。若目标频道受众不同，应重新编辑标题和说明，避免机械复制造成阅读体验下降。机器人异常时先暂停同步，再检查权限、来源设置和目标频道状态。

## 常见问题

### 是否必须把机器人设为管理员？

能否工作以及需要哪些权限取决于当前功能。涉及向频道发布内容时通常需要相应权限，但不应授予与任务无关的权限。

### 可以转载任何频道吗？

不应把技术上可见等同于拥有转载授权。转载前应遵守原创许可、频道规则、Telegram规则和适用法律。

### 为什么出现重复内容？

常见原因包括规则重复、多个任务指向同一目标，或测试任务没有停用。先暂停相关任务，再逐项核对配置。
