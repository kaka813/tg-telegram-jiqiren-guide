## 一、TG搜索机器人：查找频道、群组和资源

Telegram 内置搜索可以查找公开用户名和部分公开内容，但面对中文关键词或不清楚准确名称的情况，用户往往需要更有针对性的搜索入口。TG云搜用于按关键词搜索 Telegram 频道、群组和公开资源，并提供结果分类、收藏、历史记录等当前可见功能。

使用步骤：

1. 打开 [@TGYunSouBot](https://t.me/TGYunSouBot)；
2. 点击开始并阅读使用说明；
3. 输入主题关键词；
4. 按频道、群组或内容类型查看结果；
5. 进入结果前核对名称，不向陌生账号提交密码或验证码。

- 独立产品仓库：[TG搜索机器人｜Telegram频道、群组和资源搜索工具](https://github.com/kaka813/telegram-search-bot)
- 站内详细页面：[TG搜索机器人使用指南](./tg-jiqiren/tg-sousuo-jiqiren/)

## 二、Telegram转载机器人：同步授权内容

频道主在多个频道重复发布内容时，人工复制容易遗漏。@zhuanzai11bot 用于配置来源与目标、处理实时转载和历史回溯，并提供多种转发样式、文字规则、媒体处理和延迟设置。使用者必须拥有频道管理权限和内容授权。

使用步骤：

1. 打开 [@zhuanzai11bot](https://t.me/zhuanzai11bot)；
2. 按当前页面提示完成必要连接，任何 Session、验证码或 Token 都不得提交到 GitHub；
3. 创建任务并设置来源、目标；
4. 选择转发样式和内容规则；
5. 先检测测试，再启用实时任务。

- 独立产品仓库：[Telegram自动转载机器人｜频道内容同步与转发工具](https://github.com/kaka813/telegram-auto-forward-bot)
- 站内详细页面：[Telegram转载机器人使用指南](./tg-jiqiren/telegram-zhuanzai-jiqiren/)

## 三、TG互推机器人：管理频道合作

互推机器人用于绑定频道或群组、申请互推车队、查看候补与进度、参与曝光和管理个人资源。它帮助频道主组织合作流程，但不保证订阅增长、点击量、排名或收益。

使用步骤：

1. 打开 [@hutuiibot](https://t.me/hutuiibot)；
2. 绑定自己有权管理的频道或群组；
3. 选择对应互推项目并阅读规则；
4. 提交申请，查看候补或参与状态；
5. 与主题和受众匹配的合作方按约定执行。

- 独立产品仓库：[TG互推机器人｜Telegram频道互推与推广管理工具](https://github.com/kaka813/telegram-channel-promotion-bot)
- 站内详细页面：[TG互推机器人完整教程](./tg-jiqiren/tg-hutui-jiqiren/)

## 四、Telegram代理机器人：查看线路领取说明

@csok2bot 的公开页面提供线路领取步骤与更新提示。当前提示表明线路会按日更新，旧线路可能失效。本导航不展示代理地址、端口、密钥、二维码或完整连接链接，也不宣传绕过限制、永久在线、无限使用等效果。

使用步骤：

2. 阅读首页和指定频道的当前说明；
3. 返回机器人，从领取入口按提示操作；
4. 只在自己的 Telegram 客户端内处理连接信息；
5. 遵守当地法律、组织政策和 Telegram 规则。
